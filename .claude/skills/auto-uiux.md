**Before starting any UI or UX implementation — invoke the ui-ux-pro-max skill first.**

## Process

1. Use the Skill tool to invoke `ui-ux-pro-max`
2. Follow its design guidance for the current task
3. Then proceed with implementation

## Why this fires

The `ui-ux-pro-max` skill provides design intelligence — 67 styles, 96 palettes, 57 font pairings, 25 chart types, 13 stacks. It ensures every UI decision is informed before code is written.

## Rules

- This fires at the **start** of UI/UX work, not at the end.
- Don't skip it because the task seems small. A single button still benefits from design context.
- If the skill's guidance conflicts with `.claude/rules/design.md`, the rules file wins — it's project-specific.
