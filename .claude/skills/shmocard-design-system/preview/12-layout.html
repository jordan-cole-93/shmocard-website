Layout primitives

The two pieces every page is built from. Container sets the horizontal rail — how wide content gets and how much breathing room hugs the screen edges. Section sets the vertical rhythm — tight, deliberate breathing room between stacked blocks (we run a tight scale because we're an ecommerce site — every pixel of fold should be earning). Use these two and pages across the product feel like they belong to the same place, without anyone hand-tuning margins.
