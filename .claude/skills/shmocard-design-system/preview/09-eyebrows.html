Eyebrows & hand accent — when to use which
PILL EYEBROW
Pill eyebrow .shm-eyebrow Default for sections. Inter Tight 700, 11px, 1.6px tracking, soft pill with ember ✦. Use for section labels (FEATURES, PRICING, FAQ), category tags, step indicators. One per section, sits above the H1.
psst — five stars, easy ✦
Hand accent .shm-hand Spice — use sparingly. Shadows Into Light Two, ember (auto-honey on dark). For 1–2 word asides next to a headline, callouts inside testimonials, "limited time" / "psst" notes. Never as a section label, never for body copy, never two per section.
Decision: if it's structural (labels a section, names a category, indicates a step) → pill eyebrow. If it's conversational (winks at the reader, breaks the fourth wall) → hand accent.

Smell test: max 3 hand accents across an entire page. They're spice, not seasoning.
