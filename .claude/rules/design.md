**Direction:** Warm Professional · sans-dominant with serif italic accents. Refined, like Linear or Stripe — not neo-brutalist. Hairlines and soft shadows, not thick borders.

## Image placeholder rule

**Mascot images and card product images are always placeholders during development.** Generated by the `nano-banana` sub-agent on demand.

Mark placeholders like this:
```
[PLACEHOLDER: mascot — hero-charge pose, bold hero moment]
[PLACEHOLDER: card — CR-80 flat product shot]
```

**This rule applies ONLY to:** mascot pictures and card product pictures.
**Never applies to:** icons (Lucide SVG), decorative shapes, patterns, UI elements — those are built in code.

## Fonts

| Role | Font | Weight | Use |
|---|---|---|---|
| **Sans (dominant)** | Inter Tight | 400/500/600 | Everything — UI, body, headlines |
| **Serif (accent only)** | Fraunces italic | 400 | Once or twice per scene. Always in Ember. Never for body. |
| **Mono** | JetBrains Mono | 400/500 | Code, tokens, hex values |

**Google Fonts import:**
```
Inter+Tight:wght@400;500;600;700;800
Fraunces:ital,opsz,wght@0,9..144,400;1,9..144,400;1,9..144,500
JetBrains+Mono:wght@400;500
```

**Rule:** Fraunces italic appears once or twice per scene max, always coloured Ember (`#FF5B1F`). Never set body text in serif. Never italicise multiple consecutive words.

## Typography scale

| Style | Spec | Use |
|---|---|---|
| **Display / H1** | Inter Tight 600 · 60–68px · -0.035em · lh 1.0 | Hero headlines |
| **H2 / Section** | Inter Tight 600 · 44px · -0.025em · lh 1.05 | Section leads |
| **H3 / Card** | Inter Tight 600 · 22px · -0.015em | Card titles |
| **Eyebrow** | Inter Tight 600 · 11px · 1.6px tracking · UPPERCASE · Muted color | Section labels |
| **Lede / Body large** | Inter Tight 400 · 17px · lh 1.55 · Ink-3 | Intro paragraphs |
| **Body default** | Inter Tight 400 · 14px · lh 1.55 · Ink-2 | Body copy, UI text |
| **Stat / Number** | Fraunces 400 · 56px · -0.025em | Big metric numbers only |
| **Mono** | JetBrains Mono 400 · 12–13px | Tokens, code |

**Serif accent pattern:**
```
Turn happy customers into <em>five-star</em> reviews.
```
The `<em>` word renders in Fraunces italic, Ember colour. One accent word per headline maximum.

## Design principles

1. **Warm, not cute** — Friendly and human, never childish. A salon owner should see a real tool, not a toy.
2. **Clear over clever** — Plain language, real numbers. Microcopy explains; it doesn't perform.
3. **Mascot as accent** — Shmo lives in marketing moments (hero, mailers, packaging). Never in product UI or dashboard chrome.
4. **Hairlines & soft shadows** — Refined chrome over chunky outlines.
5. **Sans dominant** — Inter Tight carries the weight. Fraunces italic appears once or twice per scene only.
6. **Color with restraint** — Cream and paper as primary surfaces. Ember and honey as accents, not floods.

## Anti-patterns — never ship

- Thick solid borders or hard outlines on UI elements
- Solid offset shadows (no blur)
- Obviously Black or any heavy display-only font
- Fraunces for body text or multiple consecutive italic words
- Ember/orange as background flood
- Cool tones (blue, mint, teal, purple) in brand chrome
- Gradients
- Mascot inside product dashboard or UI chrome
- Generic SaaS look (thin gray borders, pale palette, no warmth)
- Inter (non-Tight) for any display use
