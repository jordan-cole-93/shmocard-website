Rules for the wave. The wave's color matches the section below it (visually it bleeds the next color upward). For the climax transition (cocoa here), use shm-wave--xl (80px) and add padding-bottom: calc(--section-py-d + --wave-height-xl) to the section above so the wave doesn't eat into content. shm-wave--lg (64px) works for ember↔marsh and other dramatic shifts; the default 18px is fine for marsh↔graham.
