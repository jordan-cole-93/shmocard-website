<interfaces>
**Endpoint:** `https://${SHOPIFY_STORE_DOMAIN}/api/2026-04/graphql.json`
**Header:** `X-Shopify-Storefront-Access-Token: ${SHOPIFY_STOREFRONT_ACCESS_TOKEN}`

**Cache tag conventions (Next.js 15):**
- `'product'` — invalidates all product reads
- `` `product:${handle}` `` — invalidates one product
- `'collection'` — invalidates collection lists
- `` `collection:${handle}` `` — invalidates one collection
- Cart fetches use `cache: 'no-store'` (mutations are not cacheable)

**Cookie contract:**
- Name: `shm-cart-id`
- Value: Shopify cart GID
- Options: `httpOnly: true, secure: true, sameSite: 'lax', maxAge: 60*60*24*14, path: '/'`

**Open-redirect guard:** before returning `checkoutUrl` to client-bound code, verify the URL host matches `*.myshopify.com` OR equals `process.env.SHOPIFY_STORE_DOMAIN`.

**Reference patterns:** RESEARCH.md Pattern 1 (fetch wrapper), Pattern 2 (server-component fetch), Pattern 3 (cart Server Action), Code Examples (queries.ts + mutations.ts).
</interfaces>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → Server Action (cart actions) | Untrusted merchandiseId, quantity, lineId crosses into mutation path |
| Shopify → Server (Storefront response) | Untrusted product description / image URLs returned to render path |
| Server → client (checkoutUrl) | Untrusted URL leaves server toward navigation |

## STRIDE Threat Register

| ID | Category | Component | Disposition | Mitigation |
|----|----------|-----------|-------------|------------|
| T-03-12-01 | Tampering | components/cart/actions.ts (addLineToCart) | mitigate | Validate `merchandiseId` matches `gid://shopify/ProductVariant/` prefix; `quantity` integer 1..99; throw on violation |
| T-03-12-02 | Tampering | components/cart/actions.ts (assertCheckoutUrl) | mitigate | Allow only hosts matching `*.myshopify.com` or `process.env.SHOPIFY_STORE_DOMAIN`; throw otherwise (open-redirect guard) |
| T-03-12-03 | Spoofing | shm-cart-id cookie | mitigate | httpOnly + Secure + SameSite=Lax + 14-day max-age (no JS access; first-party only) |
| T-03-12-04 | Information disclosure | lib/shopify/index.ts | mitigate | NEVER `console.log` env vars; never include token in error messages bubbled to client |
| T-03-12-05 | Tampering (XSS) | Shopify product description rendering | mitigate | Render `description` (plain text) only; do NOT render `descriptionHtml` (deferred per RESEARCH.md anti-patterns) |
